# Sample data

Five micro-cap symbols × 30-day windows of 1-minute OHLCV bars, sliced from the
full historical cache for the quickstart example. Each window covers a known
V5 Relaxed setup observed in the full backtest.

| File | Window | Rows | Notable |
|---|---|---|---|
| `GME.parquet`  | 2021-01-15 → 2021-02-13 | 6,814 | Jan 25 / Jan 28 wins, Jan 26 / Feb 5 losses |
| `BBIG.parquet` | 2021-08-17 → 2021-09-15 | 5,463 | Sep 1 win |
| `KOSS.parquet` | 2021-01-15 → 2021-02-13 | 2,009 | Jan 27 loss |
| `MULN.parquet` | 2022-03-01 → 2022-03-30 | 7,143 | Mar 7 win |
| `GFAI.parquet` | 2022-01-01 → 2022-01-31 |   543 | Jan 14 win |

Total: ~22K bars, ~430 KB compressed. Columns: `timestamp` (UTC, naive),
`open`, `high`, `low`, `close`, `volume`, `vwap`, `symbol`.

Regenerated by [scripts/build_sample_data.py](../../scripts/build_sample_data.py).
